# Papua New Guinea Church Website Template

This project is a modern, mobile-friendly, and PNG-culture-inspired website template for churches and Christian ministries in Papua New Guinea. The template is designed for easy customization, accessibility, and fast deployment. It uses only HTML, CSS, and minimal JavaScript, making it ideal for hosting on any static web host or shared hosting service (such as GoDaddy).

### 1. **Installation**

- Download or clone this repository.
- Extract the files if downloaded as a ZIP.
- Open the folder in your favorite editor (VS Code, Sublime, Notepad, etc).

### 2. **Customizing Text & Pages**

- Open the HTML files (such as `index.html`, `about.html`) and replace placeholder text with your churchâ€™s information.
- Each section is clearly labeled with comments and headings for easy editing.
- You can remove, duplicate, or rearrange `<section>` blocks as needed for your ministry.

### 3. **Changing Images**

- Replace placeholder images in the `/images` folder with your own church photos (keep the same filenames for easiest results, or update the filenames in the HTML if you use different names).
- Example: Replace `placeholder-worship.jpg` with a photo from your own church service.

### 4. **Styling**

- Adjust colors, fonts, and layout in `css/styles.css`. The template uses CSS variables for easy theme changes.
- The style is inspired by PNGâ€™s colors (kundu drum red, betel nut gold, rainforest green).

### 5. **JavaScript**

- The `js/main.js` file automatically updates the copyright year.
- It also enhances mobile menu and can be expanded for form validation or other simple scripts.
